# Narraluna
Plataforma de lectura y publicación de historias desarrollada con React, Vite y Firebase. Incluye autenticación de usuarios, perfiles editables, comentarios, likes, ranking de historias y sistema de búsqueda.

## Features
- Autenticación de usuarios
- Publicación de historias
- Sistema de likes y comentarios
- Ranking de historias
- Perfiles editables
- Buscador de historias
- UI moderna y responsive

## Tecnologías
- React
- Vite
- Firebase
- Firestore
- React Router
- CSS3
