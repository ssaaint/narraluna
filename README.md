# Umbral de Historias

Plataforma de lectura y publicacion de historias desarrollada con React, Vite y Firebase. Incluye autenticacion de usuarios, perfiles editables, historias originales, obras externas traducibles, capitulos, comentarios, likes, ranking, busqueda, filtros, progreso de lectura y notificaciones.

## Funciones principales

- Publicacion de historias originales.
- Fichas de obras externas para traducciones.
- Capitulos originales y traducidos.
- Likes, comentarios y seguimiento de obras.
- Perfil con avatar por URL, estadisticas y progreso de lectura.
- Home tipo dashboard y pagina Explorar con busqueda y filtros.
- Soporte de rol administrador mediante `usuarios/{uid}.rol = "admin"`.

## Tecnologias

- React
- Vite
- Firebase Auth
- Firestore
- React Router
- CSS
